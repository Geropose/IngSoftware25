- Ejecutar launch_app
- Inicialmente se ejecuta un script que instala las dependencias

Guia de uso:
MODO OFFLINE
- Probar la detección de grupos con -> Video "PruebaGrupos" ubicado en la carpeta "/videos"
- Probar los cambios de trayectorias con -> Video "PruebaTrayectorias" ubicado en la carpeta "/videos"

MODO CAMARA
- Detecta automáticamente todas las camaras conectadas con opción e elegir cual ejecutar.

MODO ONLINE
- Únicamente dejamos que se utilicen links de YouTube para la seguridad del usuario. 
Ejemplo recomendado para detección de grupos: https://www.youtube.com/watch?v=u4UZ4UvZXrg&ab_channel=EarthCam
Ejemplo recomendado para cambios de trayectorias: https://www.youtube.com/watch?v=57w2gYXjRic&ab_channel=EarthCam


- Tanto en CAMARA como en ONLINE se genera en la carpeta base un stream_output con el video procesado.
